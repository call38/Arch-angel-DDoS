это мультивекторный сетевой фреймворк давления на сервер (DDoS).
Он объединяет сразу несколько типов истощающих нагрузок, которые применяются для тестирования, или DDoS атаки.

Какие классы атак он эмулирует
1. HTTP Flood (L7-уровень)
• создаётся огромное количество HTTP-запросов
• каждый запрос выглядит как «реальный пользователь»
• сервер вынужден:
• создавать процессы/потоки
• парсить JSON / POST
• ходить в БД
• формировать ответы

Heavy POST / Payload Exhaustion
• серверу подсовывают большие POST-тела
• backend начинает:
• выделять память под них
• парсить
• сериализовывать
• логировать
• память и процессор «жрутся» кратно быстрее, чем от обычных просмотров

3. Slow-socket (Slowloris-тип)
Это атака на соединения, не на трафик.
• серверу открывают тысячи соединений
• но не завершают запросы
• сервер вынужден держать их открытыми
• у сервера заканчиваются worker-слоты

4. UDP / Network Flood (L3/L4)
• летит большой поток «пустых» сетевых пакетов
• они не доходят до сайта — они забивают линию провайдера

5. Distributed Proxy Mode
• трафик идёт не с одного айпи
• а с сотен/тысяч прокси
• фильтры и CDN начинают (видеть много разных клиентов)
